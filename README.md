# キーバインド覚えられない無理すぎ
キーバインド覚えられ無さすぎるのでチートシート作成
## tmux
設定ファイル：`~/.tmux.conf`  
Prefix： Ctrl + G に設定
| tmux | 起動 |
| --- | --- |
| Control + g d | デタッチ |
| tmux ls | セッション一覧 |
| tmux a -t セッション名 | アタッチ |
| Control + g s => 0 or 1 | セッション一覧から選択 |
| tmux kill-server | tmuxシャットダウン |
| Control + g t | 時計を表示 |

## git-cz
設定ファイル：`~/changelog.config.js`  
対話形式でコミット：git cz
